🎯 Objectif
Créer un Pokédex interactif avec Vue.js affichant uniquement des Pokémon, avec la possibilité de jouer leur cri (son) lors de l’interaction.

🧠 Contexte
Projet front-end inspiré d’un bestiaire, où chaque Pokémon possède :
une fiche visuelle
un son unique (comme dans les jeux)


⚙️ Partie technique
🧩 Technologies
Vue.js (Vue 3)
HTML / CSS
JavaScript

🌐 API utilisée
PokéAPI
